PEN-ASSEMBLE v0.5.0
===================

**Programmable Editor Nomination — Assembly pipeline for IS110-family bridge recombinase design**

PEN-ASSEMBLE is a computational pipeline for nominating IS110-family bridge recombinase
candidates for therapeutic genome editing. It scores 1,041 designs across four strategies
(A–D) using the seven-axis PenScore formula, applies five pre-registered formal predictions,
and assembles a public design catalog.

.. note::
   All five pre-registered predictions **PASS** (5/5). Publication policy: *"PUBLISH with
   strong claim"*. See :doc:`predictions`.

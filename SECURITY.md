# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.5.x   | Yes       |
| < 0.5   | No        |

## Reporting a Vulnerability

Please do **not** open a public GitHub Issue for security vulnerabilities.

Instead, email **ahmedaneesm@gmail.com** with the subject line `[pen-assemble] Security vulnerability` and include:

- A description of the vulnerability and its potential impact
- Steps to reproduce the issue
- Any suggested mitigations

You will receive a response within 5 business days. If the vulnerability is confirmed, a patch will be issued in a new release as soon as possible.
